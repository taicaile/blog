    Executing the script,

    1
    ./script.sh

    If you don't want to start a new shell but execute the script in the current shell, you source it:

    1
    source script_name.sh

    The first line of the script determines the shell to start.

    1
    #!/bin/bash

    Debugging bash scripts

    1
2
    bash -x script.sh
# the program and output will print together

    Debugging part of the script,

    1
2
3
    set -x # activate debugging here
w
set +x # disable debugging here
